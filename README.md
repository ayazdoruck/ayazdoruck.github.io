## Kurulum

Projeyi yerel ortamınıza kurmak için aşağıdaki adımları izleyin:

1. Depoyu klonlayın:

    ```sh
    git clone https://github.com/kullanici_adi/proje_adi.git
    ```

2. Proje dizinine gidin:

    ```sh
    cd proje_adi
    ```

3. Gerekli bağımlılıkları yükleyin:

    ```sh
    npm install
    ```
