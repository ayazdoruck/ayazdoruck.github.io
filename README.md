## Kurulum

1. Depoyu klonlayın:

    ```sh
    git clone https://github.com/ayazdoruck/repo.git
    ```

2. Dizine gidin:

    ```sh
    cd repo
    ```

3. Bağımlılıkları yükleyin:

    ```sh
    npm install
    ```
